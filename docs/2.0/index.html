Welcome to Spress, a static site generator for create web pages and blog without thinking about backend. Spress don't need a database. All content is powered by flat files written in Markdown.

Warning Spress 2.0 is still under development and while it is unlikely to happen, the features and changes noted in this section are subject to change before the final 2.0.0 release.
